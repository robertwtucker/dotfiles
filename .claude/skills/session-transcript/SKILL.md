# Session Transcript

Archive the current session as a structured transcript in Robert's vault.

## Action

- Create `~/vault/3-Resources/Sessions/Claude/YYYY-MM-DD - Topic.md` (replace `Topic` with a short descriptive title for this session).
- Use `Templates/Session Transcript.md` as the structure.
- Include:
  - **Summary** at the top (2-4 sentences)
  - **Decisions Made**
  - **Action Items**
  - **Key Outputs** (files created, commits, notable artifacts)

Conventions (frontmatter, naming) are in `~/vault/_system/PARA-conventions.md` if needed.
